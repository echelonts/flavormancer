How it's built & how to read it

Method. Each molecule becomes a 2048-bit Morgan fingerprint (radius 2 — a bit for every local atom environment). Distance between two molecules is Jaccard = Tanimoto on those bits, the standard chemical-similarity measure. That high-dimensional space is projected to 2D/3D with UMAP, which preserves each molecule's local neighborhood — so dots that sit together really are built from the same substructures. The layout is deterministic (fixed seed).

How to read it. The positions come purely from structure; the colors are a flavor label laid on top. Where a color forms a clean island, structure alone predicts that flavor (e.g. sugar-like polyols cluster sweet; alkaloid scaffolds cluster bitter). The interesting molecules are the off-color dots inside another cluster — a bitter point sitting among sweeteners is a structure that breaks the trend, exactly the kind of lead worth a closer look.

Two flavor dimensions. Flip the color toggle between taste (sweet / bitter / umami / sour / salty / tasteless) and aroma (= odor — the same thing; the 16 documented descriptor classes like citrus, floral, sulfurous). Same molecules, same structural layout — you're overlaying a different sense. The both toggle merges them into one full-flavor view: each dot takes its taste color where a taste is known, otherwise its aroma color — so nearly every molecule is colored by its dominant known flavor property. Comparing taste-only vs aroma-only vs both shows where the two senses align (a sweet-tasting molecule that also reads floral) versus diverge.

Honest caveats — and the fix. The similarity (UMAP) layout has axes with no units; only local distances mean anything. When you want real, labelled axes, switch the layout toggle to property axes: the same dots replotted on molecular weight (x, in daltons) vs logP (y — hydrophobicity, oil-vs-water partitioning), with gridlines and a logP = 0 waterline. That view is a metric space — you can read a molecule's size and greasiness straight off the axes, and watch a taste/aroma color track (or not track) those physical properties. In 3D, property axes add a real third dimension: TPSA (topological polar surface area — polarity / H-bonding), so it's MW × logP × TPSA. Two lenses on the same molecules: similarity for neighbors & outliers, property axes for physical structure.

Behavior & safety

Aroma

Odor descriptors are predicted from structure; intensity comes with your data.

The predicted descriptors above come from RandomForest heads trained on public-domain odor data (HSDB), the same stack as the taste engine — so aroma is predicted for any molecule, not just ones with a documented entry. This is presence/absence (which notes apply), validated per descriptor (CV-AUROC). A scored intensity map — how strong each note is — needs expert-labeled panel data, trained on-premise, on your licensed or in-house data, never leaving your firewall.

A more robust open-source academic edition — trained on richer research odor data (the GS-LF set) — is coming soon.

Bars are model predictions; a measured tag means the molecule is in the curated data (ground truth, not a guess). The trained taste/tox heads are fit on organic molecules — carbon-free inputs are flagged out-of-domain. Salty is lookup-only, sour is a rule. Tox-assay flags are indicative, caution-only (Tox21 in-vitro models) — never a determination. Quantitative dosing / odor-activity needs odor-threshold tables (licensed or customer data), so it stays qualitative here. Substitutions rank by structural similarity (Tanimoto). Aroma isn't in the public demo — it trains on your data, on-prem. Results are leads for the bench, not verdicts.

Data sources

  • PubChem (NIH / NLM) — compound identity, common & IUPAC names, measured boiling / melting / vapor-pressure data, and documented Odor & Taste annotations. public domain
  • HSDB — Hazardous Substances Data Bank (served via PubChem) — odor descriptions and odor-detection thresholds. public domain
  • CAMEO Chemicals (NOAA) & Haz-Map — physical descriptions and odor notes. public domain
  • ChemTastesDB — curated molecular taste labels (sweet / bitter / umami / sour / tasteless). academic, cited
  • FEMA GRAS — food-safe (Generally Recognized As Safe) flavor-substance reference. public listing
  • Tox21 (NIH / EPA / NCATS / FDA) — in-vitro assay data for the caution-only structural tox cross-check. public domain
  • Curated character-impact compounds — well-documented public flavor-chemistry facts (which molecule gives a flavor its signature note).
